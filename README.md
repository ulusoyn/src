### How to Install

Create a workspace
```
random@random:~$ mkdir <project_name>
```

Open the workspace
```
random@random:~$ cd <project_name>
```

Clone the repository
```
random@random:~/<project_name>$ git clone https://github.com/ulusoyn/src.git
```

Build
```
random@random:~/<project_name>$ colcon build
```
